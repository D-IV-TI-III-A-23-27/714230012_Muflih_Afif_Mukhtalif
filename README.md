## 📖 Deskripsi Proyek
**Neural-Leaf/ ANN** adalah platform edukasi interaktif berbasis web yang memvisualisasikan arsitektur **Artificial Neural Network (ANN)**. Proyek ini menggabungkan konsep kecerdasan buatan dengan estetika *Cyber-Nature* (Emerald Green) untuk memberikan pengalaman belajar yang modern dan intuitif.

### Fitur Utama:
* **Interactive Lab**: Simulasi *Forward Propagation* secara real-time dengan log status dinamis.
* **Evolution Timeline**: Menampilkan sejarah perkembangan ANN dari tahun 1943 hingga era Deep Learning modern.
* **Educational Content**: Penjelasan mengenai mekanisme training (Backpropagation), fungsi, dan tujuan ANN.
* **Responsive Design**: Tampilan yang optimal baik di perangkat desktop maupun mobile.

---

## 🛠️ Teknologi yang Digunakan
* **HTML5**: Struktur semantik untuk konten edukasi.
* **CSS3**: Desain *Glassmorphism* dengan variabel CSS kustom untuk tema Emerald.
* **JavaScript (Vanilla)**: Logika simulasi aliran saraf dan manipulasi DOM dinamis.
* **SVG**: Ikonografi daun kustom yang tajam dan ringan.
